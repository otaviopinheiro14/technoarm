**TechnoArm**
Sistema de controle web para braço robótico utilizando ESP8266.

**Descrição**
O TechnoArm é um projeto desenvolvido como TCC do curso técnico em Mecatrônica.
O sistema permite controlar um braço robótico através de uma interface web, integrando hardware e software.

**Tecnologias utilizadas**
C++
ESP8266
HTML
CSS
JavaScript

**Funcionamento**
A interface web envia comandos para o servidor, que se comunica com o ESP8266 responsável pelo controle dos servomotores do braço robótico.

**Objetivo**
Demonstrar integração entre sistemas embarcados e aplicações web, aplicando conceitos de automação e desenvolvimento de software.
